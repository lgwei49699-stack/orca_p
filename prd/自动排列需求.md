### 自动排列的需求
#### 背景
目前是使用cli命令行在服务器进行云切片，希望实现部分特定场景下的需求，一般使用下面命令行进行切片
```
orca-slicer \
  --slice 0 \
  --debug 1 \
  --arrange 0 \
  --load-settings /tmp/machine.json \
  --load-settings /tmp/process.json \
  --load-filaments /tmp/filament.json \
  --outputdir /tmp \
  --model "abc1.stl" --model-position "50,50,0" --model-scale "1" --model-rotate "0,0,0" --model-support "0"\
  --model "abc2.stl" --model-position "50,50,0" --model-scale "1" --model-rotate "0,0,0" --model-support "0"\
  /tmp/abc1.stl \
  /tmp/abc2.stl
```

#### 目的
1.目前在前端进行模型编辑的时候可能会出现模型排列过紧导致切片失败(路径冲突)，或者模型没有居中排列，或者超出一盘进行自动分盘
2.目前对模型直接打印可能支撑过多，浪费耗材，希望实现自动排列的需求


#### 实现过程中的问题
