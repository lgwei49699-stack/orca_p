## 🔍 问题分析结果

### 症状
```
[error] GLEW error code: 1
[error] GLEW error string: Missing GL version
```

### 根本原因
通过分析日志和代码，发现：

1. **✅ 代码逻辑正确**：`src/OrcaSlicer.cpp` 已经正确实现了智能 DISPLAY 检测
   - 检测到 `DISPLAY=:101` 
   - 选择使用 Native GLX API（而非 OSMesa）
   - GLFW 窗口创建成功

2. **❌ 缺少 OpenGL 库**：GitHub Action 构建环境缺少关键依赖
   - 只安装了 `libosmesa6-dev`（软件渲染）
   - 缺少 `libgl1-mesa-dev` 和 `libglu1-mesa-dev`（硬件加速 OpenGL）
   - 导致 GLEW 无法加载 OpenGL 函数指针

3. **⚠️ GLFW 配置未知**：`deps/GLFW/GLFW.cmake` 可能禁用了 X11
   - 需要确保 `GLFW_BUILD_X11=ON`

---

## 🔧 解决方案（按优先级）

### 方案 A：修改 GitHub Action 依赖配置（必须）

**文件：`.github/workflows/build-linux.yml`**  
**位置：第 65 行附近**

当前配置：
```yaml
sudo apt-get install -y libunwind-dev libosmesa6-dev
# 安装 X11 开发库用于 GLFW X11 支持
sudo apt-get install -y libx11-dev libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev xvfb
```

**需要改为**：
```yaml
sudo apt-get install -y libunwind-dev libosmesa6-dev
# 安装 OpenGL 核心库（关键！）
sudo apt-get install -y \
  libgl1-mesa-dev \
  libglu1-mesa-dev \
  mesa-common-dev \
  libegl1-mesa-dev
# 安装 X11 开发库用于 GLFW X11 支持  
sudo apt-get install -y libx11-dev libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev xvfb
```

**原因说明**：
- `libgl1-mesa-dev`：提供 OpenGL API（glGetString, glViewport 等）
- `libglu1-mesa-dev`：提供 GLU 工具函数
- `mesa-common-dev`：Mesa 通用头文件
- `libegl1-mesa-dev`：EGL 支持（可选但推荐）

---

### 方案 B：检查并修改 GLFW 编译配置（如果方案 A 无效）

**文件：`deps/GLFW/GLFW.cmake`**

**需要检查的内容**：
```cmake
if(CMAKE_SYSTEM_NAME STREQUAL "Linux")
    set(_glfw_build_x11 "-DGLFW_BUILD_X11=ON")  # ← 必须是 ON
    set(_glfw_disable_wayland "-DGLFW_BUILD_WAYLAND=OFF")
    set(_glfw_use_osmesa "-DGLFW_USE_OSMESA=ON")
```

**如何检查**（在 Mac 本地）：
```bash
cat deps/GLFW/GLFW.cmake | grep "GLFW_BUILD_X11"
```

如果显示 `OFF`，需要改为 `ON`，然后：
1. 清理缓存（GitHub Action 第 76-83 行已配置）
2. 重新构建（会自动执行）

---

### 方案 C：添加构建验证步骤（推荐）

**文件：`.github/workflows/build-linux.yml`**  
**位置：第 99 行之后添加**

```yaml
    - name: Verify OpenGL environment
      shell: bash -e {0}
      run: |
        echo "=== 验证 OpenGL 库 ==="
        ldconfig -p | grep libGL || echo "⚠️ libGL 未找到"
        ldconfig -p | grep libGLU || echo "⚠️ libGLU 未找到"
        
        echo ""
        echo "=== 验证 GLFW 配置 ==="
        if [ -f "deps/GLFW/GLFW.cmake" ]; then
          grep "GLFW_BUILD_X11" deps/GLFW/GLFW.cmake || echo "⚠️ X11 配置未找到"
        fi
```
